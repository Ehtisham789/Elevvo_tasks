**Elevvo Frontend Development Tasks**

This repository contains a collection of frontend development tasks completed as part of the Elevvo Internship Program.
Each task focuses on strengthening core frontend concepts such as responsive design, UI development, component structure, state management and API integration.

**Tasks Overview**

**Task 1: Collapsible Sidebar**  
Built an interactive sidebar that can be toggled open and closed with smooth animations.

**Key Features:**
+ Logo placeholder
+ Navigation links with icons
+ Toggle functionality
+ Clean layout with animation
+ Mobile responsiveness (Bonus)

**Concepts Covered:**  
HTML | CSS | JavaScript | Component Layout | Animation | Toggle Logic

---

**Task 2: Contact Form**  
Created a responsive contact form designed to work across all screen sizes.

**Key Features:**
+ Input fields: Full Name, Email, Subject, Message
+ Structured layout with proper spacing
+ Mobile friendly design
+ Basic form validation using JavaScript

**Concepts Covered:**  
Forms | Responsive Design | CSS Styling | Input Validation

---

**Task 3: One Page Task App Website**  
Designed a modern landing page for a fictional productivity app TaskFlow.

**Key Sections:**
+ Hero section with CTA
+ Features section
+ User testimonials
+ Pricing plans
+ Footer with social links
+ Scroll based animations 

**Concepts Covered:**  
Component Design | Responsive Layout | UI Structuring | React Basics

---

**Task 4: Personal Blog Homepage**  
Developed a blog homepage using a card based layout.

**Key Features:**
+ Blog header
+ Post cards (image, title, description, date)
+ Category filtering
+ Post limit per page
+ Search functionality

**Concepts Covered:**  
Props & State | Conditional Rendering | Filtering | Responsive Grid

These tasks were designed to simulate real world frontend development challenges and helped strengthen practical skills in building responsive interfaces, managing application state, integrating APIs, and creating user friendly layouts.

This repository reflects my learning journey and hands on experience gained during the Elevvo Internship Program.
